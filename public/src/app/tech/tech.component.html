Angular was the main technology used for this website. Angular is a powerful front end technology that allows for fully functional single page applications. The website was broken into many seperate 'components' and each component has it's own HTML, CSS, and TypeScript files (TypeScript is similiar to JavaScript). When the browser clicks a link, rather than routing to a seperate page in order to display different content, I am merely adjusting the appearance of a single page using these seperate components.
